# GitHub Secrets Example

`Settings -> Secrets and variables -> Actions`에 아래 값을 등록합니다.

## Front Secrets

### `VERCEL_TOKEN`
```text
your-vercel-token
```

### `VERCEL_ORG_ID`
```text
your-vercel-org-id
```

### `VERCEL_PROJECT_ID`
```text
your-vercel-project-id
```

## Notes
- 이 파일은 예시 문서입니다.
- 실제 secret 값은 GitHub Actions Secrets에만 등록하고 Git에는 올리지 않습니다.
